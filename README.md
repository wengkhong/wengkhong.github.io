# Weng Khong Lim – Personal & Lab Website

This repository contains the source code for my personal academic website.

## 🌐 Live site
https://wengkhonglim.github.io  
(custom domain later)

## 🧱 Tech stack
- Astro (static site generator)
- Markdown content collections
- GitHub Pages (hosting)

## 🚀 Development

Run locally:

```bash
npm install
npm run dev

Build:

npm run build

📁 Structure
src/pages/ – main pages (About, Projects, Publications, etc.)
src/content/ – structured content (markdown)
src/components/ – reusable UI components

📦 Deployment
Deployed automatically via GitHub Actions to GitHub Pages.
